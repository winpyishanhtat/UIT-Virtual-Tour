# 🎓 UIT Virtual Tour System (Vizard VR)

## 📌 Project Overview
The **UIT Virtual Tour System** is a 3D interactive virtual environment developed using **WorldViz Vizard** and Python.  
This project simulates the campus structure of the **University of Information Technology (UIT)**, allowing users to explore buildings, bridges, halls, and campus infrastructure in a realistic virtual space.

The system demonstrates computer graphics concepts, 3D modeling, lighting simulation, and virtual navigation.

---

## 🏫 Features

- ✅ Realistic 3D campus visualization
- ✅ Multi-floor academic buildings
- ✅ Glass hall and bridge connections
- ✅ Detailed stair and railing systems
- ✅ Campus landscape with roads and fences
- ✅ Realistic lighting and materials
- ✅ Interactive first-person navigation

---

## 🛠️ Technologies Used

- **Python**
- **WorldViz Vizard**
- Computer Graphics
- 3D Geometry Modeling
- Lighting & Rendering Simulation
